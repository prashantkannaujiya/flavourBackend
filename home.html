Welcome to Flavour

This is the backend site of Flavour

Currently, there is no administrator support. But, there are a number of routes you can use to feed dummy data to your online food site or whatever. It got decent collection of food items and corresponding categories to use for testing frontend for its relatable features.

It uses dummy data from mealdb and provides support for dummy payments using Razorpay, including support for cart. Creating user account is enabled too.

Example

fetch('https://flavour-backend.vercel.app/fetch/Chicken/-1')

.then((res)=>{res.json()})

.then((data)=>{'Your code here'})

Routes available

GET - /fetch/categoryName/priceRange

Set priceRange as -1 to have no price pmit

Set priceRange as 0 for less than $50

Set priceRange as 50 for in between $50 and $100.

Set priceRange as 100 for greater than $100.

Put categoryName exactly same as in menu

POST - /register

Register your customer

POST - /login

Login your customer using credentials. It provides a login token on successfp completition.

GET - /auth/:token

Enables auto login using token for a user logged in previously.

GET - /addToCart/:product/:user/

Add products to cart of a user using product id and user id.

GET - /cartupdate/:productid/:userid/:quantity

Update quantity of a product in cart.

GET - /cartProduct/:user

Fetch all the products in a user's cart, provided, user is logged in.

GET - /cartRemove/:user/:id

Remove a product from a cart by providing user id and product id as params.

GET - /searchFood/:search

Search for a particpar food item, category or anything that matches the description of your food item.

GET - /order

Place order for payment to Razorpay payment gateway.

GET - /success

Verify payment for any discrepency and confirm it.
