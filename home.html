Welcome to Flavour

This is the backend site of Flavour

Currently, there is no administrator support. But, there are routes you can use to feed dummy data to your online food site. It includes a decent collection of food items and categories for testing frontend features.

It also supports dummy payments via Razorpay, including cart functionality and user account creation.

Example

fetch('https://flavour-backend.vercel.app/fetch/Chicken/-1')
.then((res) => res.json())
.then((data) => 'Your code here');
Routes Available

GET - /fetch/categoryName/priceRange

Set priceRange to -1 for no limit, 0 for below $50, 50 for $50-$100, or 100 for above $100. Ensure categoryName matches menu exactly.

POST - /register

Register your customer.

POST - /login

Login with credentials, returns a token upon successful login.

GET - /auth/:token

Auto login with token for previously logged-in users.

GET - /addToCart/:product/:user/

Add product to cart using product and user IDs.

GET - /cartupdate/:productid/:userid/:quantity

Update product quantity in cart.

GET - /cartProduct/:user

Retrieve all items in a userâ€™s cart (login required).

GET - /cartRemove/:user/:id

Remove product from cart by user and product IDs.

GET - /searchFood/:search

Search for food items by name, category, or description.

GET - /order

Initiate order for payment via Razorpay.

GET - /success

Confirm payment to resolve any discrepancies.
